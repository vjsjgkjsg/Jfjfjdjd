  КАК ОТКРЫТЬ САЙТ (ЛОКАЛЬНО)
─────────────────────────────────────────

Способ 1 — просто открыть файл:
  1. Разархивируйте ZIP в любую папку на компьютере.
  2. Откройте файл index.html двойным щелчком мыши.
  3. Сайт откроется в браузере (Chrome, Firefox, Edge).
  
  ВАЖНО: все три файла (index.html, style.css, script.js)
  должны находиться В ОДНОЙ ПАПКЕ.


Способ 2 — через локальный сервер (рекомендуется):
  Если у вас установлен Python:
  
    Для Python 3:
    > cd папка-с-сайтом
    > python -m http.server 8080
    
    Затем откройте в браузере: http://localhost:8080
    
  Если у вас установлен Node.js:
    > npx serve .
    
    Затем откройте указанный адрес в браузере.

─────────────────────────────────────────
  НАСТРОЙКА ДЛЯ ВАШЕЙ ОРГАНИЗАЦИИ
─────────────────────────────────────────

1. ССЫЛКА НА TELEGRAM-ЧАТ:
   В файле index.html найдите строки с href="https://t.me"
   и замените на реальную ссылку вашего чата.
   Пример: href="https://t.me/+AbCdEfGhIjKlMnOp"

2. ДОБАВЛЕНИЕ РЕАЛЬНЫХ ФАЙЛОВ ДЛЯ СКАЧИВАНИЯ:
   Положите PDF и PPTX файлы рядом с index.html.
   В файле script.js найдите функцию fakeDownload()
   и замените её на настоящую ссылку для скачивания.
   
   Пример:
     window.fakeDownload = function(filename) {
       window.open(filename, '_blank');
     };

3. СМЕНА КОНТАКТОВ / ЛОГОТИПА:
   Откройте index.html в текстовом редакторе (Notepad++,
   VS Code) и отредактируйте нужные строки.

4. ПУБЛИКАЦИЯ В ИНТЕРНЕТ:
   Загрузите все три файла на хостинг (например,
   GitHub Pages, Netlify, или школьный сервер).
